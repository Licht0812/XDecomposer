# XRD分离与多相分类评估指标详解

这些指标评估了 XRD 分离或多相分类模型在不同维度（结晶学匹配度、混合物相存在判断、以及具体物相鉴定）上的表现。根据代码库中的逻辑，它们的详细计算方式和定义如下：

## 1. `id_acc_topk` (Top-K 物相鉴定准确率)

**定义：** 衡量模型为每个“真实存在的物相”预测出的参考数据库候选物相（Top-K 相似度排列）中，是否包含了该真实的物相 ID。

**代码逻辑（基于预测和数据库特征余弦相似度检索）：**
- **活跃判断**：如果一个物相的目标特征能量 `target_energy > 1e-4`，则认为该物相在样本中真实存在（Active Phase）。
- **预测召回**：模型针对分离出来的特征表示，计算其与参考数据库（Reference Bank）中所有光谱的余弦相似度，并取出最相似的前 K 个物相 ID（`pred_topk_ids`）。
- **匹配计算**：如果该真实活跃物相的 Target ID 在预测出的 Top-K ID 列表内出现，命中数（`cum_match`）判定为 1。该指标最终是命中数除以所有真实活跃物相的总数量。
- **意义**：这本质上是一个基于表征学习的信息检索（Information Retrieval）或指纹鉴定指标。

## 2. `rwp` (Residual Weighted Profile, 晶体学标准残差因子)

**定义：** Rwp 是粉末 X 射线衍射（XRD）精修（Rietveld Refinement）和结构分析中最核心的拟合优度指标。它衡量了模型重构的 XRD 图谱（`pred_pattern`）与真实衍射图谱（`target_pattern`）在轮廓上的整体差异。

**数学公式（来源于 `src/utils/metrics.py`）：**
$$ Rwp = \sqrt{\frac{\sum w_i (y_{obs\_i} - y_{calc\_i})^2}{\sum w_i y_{obs\_i}^2}} $$

在实现中：
- $y_{calc\_i}$ 是预测的图谱强度值，由于 XRD 通常没有非常严格的权重阵，代码经过了归一化并默认 $w_i = 1$。
- **意义**：Rwp **越低**（越接近 0），说明预测图谱和真实图谱在峰位、峰宽、峰强的整体连续轮廓上匹配得越完美。

## 3. `act_*` (活性/存在性分类评估簇)

此簇涵盖 `act_f1`, `act_recall`, `act_precision`, `act_exact_match`。它们将“图中是否包含该物相”视为了一个**多标签二分类问题**（Multi-label Binary Classification）。

**基础阈值逻辑（来源于测试脚本）：**
- **真实标签（Ground Truth）**：计算该物相真实衍射特征模式的能量分布，如果能量 `(targets ** 2).sum(dim=-1) > 1e-6`，则认为 `target_is_active = True`（该相存在）。
- **预测标签（Prediction）**：模型输出存在的逻辑值（logits），通过 Sigmoid 激活函数后，如果 `torch.sigmoid(aligned_act_logits) > 0.5`，则模型认为该物相预测出对应为 `act_pred = True`。

基于这两个布尔类型的张量，各项统计定义如下：

### 3.1 `act_recall` (召回率)
- **公式**：`TP / (TP + FN)`
- **定义**：真实存在的物相中，模型成功预测它存在的比例。反映模型**遗漏目标的概率（是否漏报）。**

### 3.2 `act_precision` (精确率)
- **公式**：`TP / (TP + FP)`
- **定义**：模型预测判断为存在的物相中，实际上真正存在的比例。反映模型**误报的概率（是否乱报）。**

### 3.3 `act_f1` (F1 调和分数)
- **公式**：`2 * Precision * Recall / (Precision + Recall)` 
- **定义**：精确率和召回率的调和平均数，能整体反映模型在长尾或多组分样本中，存在性分类的整体鲁棒性。

### 3.4 `act_exact_match` (全匹配率 / 子集精确度)
- **代码运算**：`(act_pred == target_is_active).all(dim=1).float().mean()`
- **定义**：严苛的整体正确率。要求单个多相混合物样本中，**所有组分的“存在”和“不存在”状态被模型 100% 完全全对预测**（连带正确的和不该有的都不能越界），该样本才算完全 Match。这个指标通常要远低于 F1，是多相定性分析中最具挑战的指标。
